Extension Killer

What does it do?

The extension killer temporarily disables any extension until the computer is shut down and signed back into.

How do I do it?

First, you need to get the extension's ID. Here's how:

  1. Open Chrome and go to chrome://extensions/
  2. Find the extension you want the ID for and enable "Developer mode" in the top right corner.
  3. You'll now see the extension's ID displayed below it. Copy it.

Now that you have the extension's ID, follow these steps:

  1. Go to chrome-extension://PUT-YOUR-EXTENSION-ID-HERE/manifest.json (replace PUT-YOUR-EXTENSION-ID-HERE with the actual ID), bookmark it, and name it A.
  2. Next, go to chrome://kill, bookmark it, and name it B.
  3. Now, go to chrome://hang, bookmark it, and name it C.
  4. Close all the tabs and open them in the following order: A, then B, reload the page, and finally C.

That's it!

What does my teacher see?

It varies from extension to extension, but for most, it will look like you're offline.

How can I avoid getting in trouble?

I recommend deleting the bookmarks after use and shutting down the computer every once in a while in case an admin tries to check out your computer.
